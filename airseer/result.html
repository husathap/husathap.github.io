Layman's Term

Model 1 performs worse than Model 2 close to the Arrival Staircase. Model 1 performs well near Gate 7, 8. Model 2 performs poorly near the lounge. The sample sizes may be too small; the user may need better data collection.

Layman's Term is a very high-level interpretation of the visualization for a quick and dirty analysis. It is not necessarily accurate. The user is encouraged to apply their own expertise as well as additional information below.

Technical Details

R-squared

This effect size that we primarily use for analysis. The effect size is global; that is, it assumes the models to work equally well in all areas. While we can add a zone term into the models, the zone term does not necessarily mean that the models will work equally well in all areas.
  • Model 1: 60%
  • Model 2: 70%

ANOVA

This is a novel type of ANOVA that tests whether the models' R-squared values differ between each zone or not. If there is an interaction between Model and Zone, then it may be a good idea to create zone-specific models. If there is only the effect of the Model, then the best model maybe the one with the highest F. If there is only the effect of the area, it suggests that all models work equally well but we should create zone-specific models.
